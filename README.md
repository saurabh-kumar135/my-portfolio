## Projects Featured

### HavenTo

Accommodation booking platform built with MERN stack. Has features like property listings, user auth, favorites, and booking management.

### StudyMate

AI study platform using Google Gemini. Includes AI tutor, document summarization, quiz generation, and study analytics.

### Brain Tumor Detection

Medical AI system for detecting brain tumors from MRI scans. Uses deep learning with TensorFlow/Keras.

Or use a local server:

```bash

```
